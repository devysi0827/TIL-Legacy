# sql 고득점 kit

1. 최댓값 구하기

```mysql
SELECT MAX(DATETIME) FROM ANIMAL_INS;
```



2. 최솟값 구하기

```mysql
SELECT MIN(DATETIME) FROM ANIMAL_INS;
```



3. COUNT

```mysql
SELECT COUNT(DATETIME) FROM ANIMAL_INS;
```



4. DISTINCT

```mysql
SELECT COUNT(DISTINCT NAME) FROM ANIMAL_INS
```
