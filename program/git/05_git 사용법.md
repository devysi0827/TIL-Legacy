##   git 업로드 방법

cd 

cd submission : 과제 제출 폴더로 들어간다

git add . : stage에 올린다

git commit -m '메세지 내용' : 내용 업로드

git push origin master :

git log : git log를 확인한다







## 기타
- git amend : 메세지 수정 (아직 자세히 배우지 않음)
- git init : 폴더를 저장소화 (master)를 붙임
  - ~폴더는 저장소화되면 안됌
  - 그 하위폴더도 저장소화화면 안됌
-  ctrl c(^c) : 내용 취소
- $: 준비완료 상태, >: EOL, 입력미완상태
