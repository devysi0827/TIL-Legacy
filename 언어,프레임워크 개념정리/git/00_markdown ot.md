# Markdown OT



## 명령어

- ctrl +'/' : 마크다운 기본 형식으로 페이지 전환

- `~`

  home 폴더를 의미

  위치는 일반적으로 /C/Users/[계정명]

- / : root (?) 

- cd :홈으로 이동

  - .   : 현재 위치 표시

	- .. : 전 단계 위치로 이동

- `-`: 옵션

  - -f  : 강제실행

- mkdir  : 폴더 생성

- rm : 파일 삭제

- ls : 현재 폴더의 리스트 표시
  
- tab : 자동완성

- tab x2 : 자동완성 선택지 제공

- code + . : 현재 폴더에서 vs code 실행

- `*` : 와일드 카드

- | | | 표 만들기

  - 이 상태로 ctrl+enter 행 추가

##  기본 양식

- -/+/*+space 키 번호 없는 동그라미(계속 사용시 계속 생성)
- num. 넘버
- *기울임*  (*or_ text *or_)
- **굵게**(* *or_ text **or__)
- 수평선 -_* 중 같은 거 세 개 나열
